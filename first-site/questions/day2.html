1. 链接是什么概念，对应什么标签

    2. 我觉得标签是指一个捷径，它可以是一个字、一段文字或是一张图片，这些特定的内容通常会指向一个新的文档或是当前文档的某一指点部分
    对应的标签是<a>标签

  2. 常用的标签有哪些，都用在什么场景

    1. div 定义文档中的分区或节
    2. p 定义段落
    3. h1~h6 定义html标题
    4. table 定义表格
    5. ol 定义有序列表
    6. ul 定义无序列表
    7. style 定义文档的样式信息
    8. form 定义供用户输入的表单

  3. ul、ol、li与dl、dt、dd的区别

    4. d系列表是自定义列表，其中dl表示定义一个列表，dt定义列表中的项目，dd对项目进行描述
    而ul是定义无序列表，ol定义有序列表，li定义一个列表项

  4. img中的src和alt属性的作用

    5. src指明图片所在文件路径
    alt是对图片进行一个文字描述，当出现络错误、内容被屏蔽或链接过期时会显示出来

  5. img中的title属性会产生什么效果

    6. 当网页上的图片被加载完成后，鼠标移动到上面去，会显示这个图片指定的属性文字，以对图片信息进行补充性说明

  6. 一个页面只能有一个 h1 吗？为什么？

    7. 不一定，有多个和h1标签网页还是了够正常显示，但是严格按照文档标准进行编码的话，最好只出现一个h1标签

    8. div是块级元素，span是行内元素

  8. a 标签的 tagert 属性值有哪些？对应作用是什么？

    9. tagert属性指定在何处显示链接的资源，共有4个属性值
    • _self: 当前页面加载，即当前的响应到同一 HTML 4 frame（或 HTML5 浏览上下文）。此值是默认的，如果没有指定属性的话。
    • _blank: 新窗口打开，即到一个新的未命名的 HTML4 窗口或 HTML5 浏览器上下文
    • _parent: 加载响应到当前框架的 HTML4 父框架或当前的 HTML5 浏览上下文的父浏览上下文。如果没有 parent 框架或者浏览上下文，此选项的行为方式与 _self 相同
    • _top: IHTML4 中：加载的响应成完整的，原来的窗口，取消所有其它 frame。 HTML5 中：加载响应进入顶层浏览上下文（即，浏览上下文，它是当前的一个的祖先，并且没有 parent）。如果没有 parent 框架或者浏览上下文，此选项的行为方式相同_self
